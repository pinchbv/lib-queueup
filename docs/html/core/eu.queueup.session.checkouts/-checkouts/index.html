abstract suspend fun createCheckout(date: Instant, flexibleDate: Boolean = false, lines: List<Line>, externalReference: String?, idempotencyConfig: IdempotencyConfig? = null): Checkout

Creates a checkout (reservation) for one or more products at a specific merchant. The checkout expires if not confirmed within its TTL. The SDK manages an Idempotency-Key internally: repeated calls with identical input within a short window reuse the same key so the backend collapses duplicates.
